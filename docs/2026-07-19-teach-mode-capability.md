# Teach Mode Capability

A teacher can run a generated lesson from one device, check off objectives, and save a private reflection locally in that browser.

## Constraints

- No names, emails, uploads, grades, or student responses are collected.
- Lesson progress and teacher reflection stay in browser `localStorage` on that device.
- Clearing browser site data clears saved Teach Mode sessions.

## Setup

1. Rebuild, generate a lesson, and choose **Teach on this device**.
2. Lesson progress and the reflection are saved automatically in the current browser.
